Bookmarklet? WTF o_O

Hey yo! Do you know that we can execute script from the Chrome's addressbar?

Paste this in your address bar: ( function() { alert('Wazzzzzzup!'); } )() and write in front of it javascript: to get javascript:( function() { alert('Wazzzzzzup!'); } )() and press Enter.

Neat, right?)

More over what we can do is to create a link with the href attribute with the value of our script-link.

<a href="javascript:( function() { alert('How you doin? :D'); } )()">Press this script-link</a>

Press this script-link.

Of course calling alert is the basic and the simplest thing we can do. It will be great competition to write something more complex using one string in the href attribute. Do we need this? Not at all ;)

We can create a simple script for the href attribute which will inject our complex and awesome script from somewhere (for example from your repo on Github):

      (function () {
        const script_url = 'https://perioad.github.io/bookmarklet/bookmarklet.js';
        const scriptElement = document.createElement('script');
        scriptElement.src = script_url;
        document.body.append(scriptElement);
      })();

Let's format it in one line and paste it in a link (don't forget about javascript:):

<a href="javascript:(function () {const script_url = 'https://perioad.github.io/bookmarklet/bookmarklet.js';const scriptElement = document.createElement('script');scriptElement.scr = script_url;document.append(scriptElement);})()">Press this script-link</a>

And here the content of my bookmarklet.js:

      (function () {
        const paragraphs = document.getElementsByTagName('p');
        for (let paragraph of paragraphs) {
          paragraph.style.background = 'coral';
        }
      })();

And here our script-link.

So right now if press the link above it will execute the script in the addressbar which pastes a script element on the page which will execute my bookmarklet.js script from Github which changes the background color of all p tags on a page.

Moreover you can add this script-link to your bookmarks and change the background color of all p tags on any page!
