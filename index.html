Hey yo! Do you know that we can execute script from the Chrome's addressbar?

Paste this in your address bar: ( function() { alert('Wazzzzzzup!'); } )() and write in front of it javascript: to get javascript:( function() { alert('Wazzzzzzup!'); } )() and press Enter.

Neat, right?)

More over what we can do is to create a link with the href attribute with the value of our script-link.

<a href="javascript:( function() { alert('How you doin? :D'); } )()">Press this script-link</a>

Press this script-link.

Of course calling alert is the basic and the simplest thing we can do. It will be great competition to write something more complex using one string in the href attribute. Do we need this? Not at all ;)

We can create a simple script for the href attribute which will inject our complex and awesome script from somewhere. For example from your repo on Github.
